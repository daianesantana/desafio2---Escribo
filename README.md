1. Tecnologias Utilizadas
Node.js
Expresso.js
Sequelize 
MySQL
JSON Web Token (JWT) para autenticação
Bcrypt para criptografia de senhas

2. Instale as dependências:
cd desafio2
npm install
Execute o seguinte comando para iniciar o servidor:
npm start
